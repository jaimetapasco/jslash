jslash.Canvas(canvasId)
Wrapper to HTMLCanvasElement. It can draw any drawable object. It is used to handle HTML-defined canvas or to create one if it's needed.
Parameters:
{string} canvasId Optional
Identifier of the HTML-defined Canvas Element to use.
